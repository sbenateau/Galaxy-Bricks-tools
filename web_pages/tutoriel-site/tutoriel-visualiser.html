Visualiser mes données

 

Vous disposez d'un d'outil pour représenter vos données. Cette étape vous permet de représenter clairment vos résultats et mettre en évidence les différences ou les relations entres variables par exemple.

Il existe deux outils. Le premier traite les tableaux avec plusieurs lignes (la plupart des utilisations).

Représenter les données

 

Le deuxième traite les données avec une seule ligne, issues de Opération sur des colonnes par exemple.
